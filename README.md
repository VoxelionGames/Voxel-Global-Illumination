# Voxel-Global-Illumination
A Voxel-Based Diffuse Global Illumination Solution

Hello everyone, this is a new global illumination solution, which is original to me—a hybrid variant of some known algorithms. The whole algorithm can be seen as a radiosity method in voxel face space, but with some differences.
